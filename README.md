## 🚀 Featured Projects

### 🔌 [Ollama USB Toolkit (Claw Edition)](https://github.com/musagithub1/ollama-usb-toolkit)
A plug-and-play, **fully offline AI agent** that runs from a USB drive on Windows, macOS, and Linux — no installation, no internet, zero telemetry. Built a single-agent runtime with a custom Python Agent API, a sandboxed skill/tool-execution loop (function calling, file I/O, shell), and a static web UI, all powered by local Ollama models.
> `Python` · `Ollama` · `Local LLM Agents` · `Function Calling` · `Cross-Platform`

### 🎯 [HireSense AI](https://github.com/musagithub1/HireSense_AI)
An **adaptive AI interview platform** that personalizes questions from your resume + job description using **RAG**, reads your emotions in real time via webcam, and adapts its tone and difficulty to your stress level. Includes voice I/O, analytics dashboards, a live interview copilot, and secure Firebase Google SSO.
> `Streamlit` · `LangChain` · `OpenRouter` · `TensorFlow.js` · `RAG` · `MediaPipe` · `Firebase`

### 🧠 [Viva Defense — Face Sensor](https://github.com/musagithub1/Viva-Defense-Face-Sensor)
A real-time facial-expression sensor that classifies whether a person looks **Confident or Stressed** during a viva/interview. Re-engineered the FER2013 dataset into a binary problem and trained a **custom CNN with Spatial Attention Modules**, achieving **93.49% AUC** and 85% accuracy with mixed-precision training.
> `TensorFlow` · `Keras` · `Custom CNN` · `Spatial Attention` · `OpenCV` · `TFLite`

### 🚗 [License Plate Detection (OpenCV + YOLOv8)](https://github.com/musagithub1/license-plate-detection-opencv-yolov8)
A **production-grade, real-time license plate detection system** written in **C++** using OpenCV's DNN module and an ONNX-optimized YOLOv8 model. Supports images, video, webcam, and IP-camera streams with **92.3% mAP@0.5** and high-FPS inference.
> `C++` · `YOLOv8` · `OpenCV DNN` · `ONNX` · `CMake` · `Real-Time Inference`

### 👁️ [Face Detection Project](https://github.com/musagithub1/FACE_DETECTION_PROJECT)
A comprehensive **face analysis toolkit** in Python/OpenCV with face detection, age/gender prediction, emotion & smile analysis, head-pose estimation, glasses detection, facial symmetry scoring, privacy blurring, filters, and batch processing — usable via both a Tkinter GUI and a full CLI.
> `Python` · `OpenCV` · `NumPy` · `Tkinter` · `Haar Cascades`
